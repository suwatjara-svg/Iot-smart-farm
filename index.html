หลักสูตรการติดตั้งเครื่องรดน้ำระบบอัตโนมัติ

Smart Farmer · Smart Farming · เกษตรอัจฉริยะ

สถาบันพัฒนาฝีมือแรงงาน 4 ราชบุรี

หลักสูตร: การติดตั้งเครื่องรดน้ำระบบอัตโนมัติ
วิทยากร: นายสุวัฒน์ เจริญพิบูลย์ · นักวิชาการพัฒนาฝีมือแรงงานชำนาญการ

โมดูลการเรียนรู้

Module 1
Smart Farmer คืออะไร
แนะนำแนวคิด Smart Farmer, Smart Farming, ตัวอย่างการประยุกต์ใช้ในฟาร์มจริง และเหตุผลที่พืชเติบโตดีกว่าระบบเกษตรแบบเดิม
เข้าสู่บทเรียน
Module 2
ไฟฟ้าเบื้องต้นสำหรับ Smart Farm
เรียนรู้ประเภทของไฟฟ้า ชนิดของสายไฟ อุปกรณ์ไฟฟ้าที่จำเป็น การใช้มิเตอร์ คีมปอกสาย คีมย้ำ และหลักความปลอดภัยในฟาร์ม
เข้าสู่บทเรียน
Module 3
ระบบสื่อสาร IoT (WiFi, Zigbee, Z‑Wave, BLE, LoRa)
เปรียบเทียบเทคโนโลยีสื่อสารสำหรับ IoT ในงานเกษตร: ระยะทาง ความเร็ว การใช้พลังงาน และตัวอย่างการเลือกใช้ในระบบรดน้ำและฟาร์มขนาดต่าง ๆ
เข้าสู่บทเรียน
Module 4
การใช้ Sonoff ควบคุมอุปกรณ์ไฟฟ้า
การต่อ Sonoff แบบ 3 รูปแบบ: ควบคุมโหลดโดยตรง, เพิ่มสวิตช์ Auto / Off / Man, และการใช้ร่วมกับ Magnetic Contactor สำหรับปั๊มน้ำขนาดใหญ่
เข้าสู่บทเรียน
Module 5
ตัวอย่างวงจรทดลอง (ทดลอง 1–3)
รวมตัวอย่างวงจรจากการทดลองจริง: เปิด–ปิดหลอดไฟ, โหมด Auto / Off / Man, และวงจรที่ใช้ Magnetic Contactor ในระบบรดน้ำอัตโนมัติ
เข้าสู่บทเรียน
Module 6
วงจรจำลอง Interactive
ทดลองกดสวิตช์ ดูการไหลของไฟฟ้าตามสาย ผ่านวงจรจำลองแบบ Interactive เหมาะสำหรับใช้สอนและอธิบายหลักการทำงานให้ผู้เรียนเข้าใจง่าย
เข้าสู่บทเรียน
Module 7
แบบทดสอบและสรุป
แบบฝึกหัดและแบบทดสอบสรุปเนื้อหา Smart Farmer, ไฟฟ้าเบื้องต้น, ระบบสื่อสาร IoT, การใช้ Sonoff และวงจรทดลอง เพื่อประเมินความเข้าใจของผู้เรียน
ทำแบบทดสอบ
