Smart Farmer · โดย สถาบันพัฒนาฝีมือแรงงาน 4 ราชบุรี

Module 1: Smart Farmer คืออะไร

เรียนรู้แนวคิดเกษตรอัจฉริยะ ประเภทของ Smart Farming และข้อดีที่ทำให้พืชเจริญเติบโตดีกว่าเดิม

Module 2: ไฟฟ้าเบื้องต้นสำหรับ Smart Farm

พื้นฐานไฟฟ้า ประเภทของไฟฟ้า ชนิดสายไฟ อุปกรณ์ไฟฟ้า และความปลอดภัยในการใช้งานในฟาร์ม

Module 3: ระบบสื่อสาร IoT สำหรับ Smart Farm

เปรียบเทียบเทคโนโลยี WiFi, Zigbee, Z-Wave, BLE และ LoRa สำหรับการเชื่อมต่ออุปกรณ์ในฟาร์ม

Module 4: การใช้งาน Sonoff ในระบบรดน้ำอัตโนมัติ

เรียนรู้การต่อ Sonoff แบบต่าง ๆ เพื่อควบคุมอุปกรณ์ไฟฟ้าในฟาร์ม

Module 4.1: แบบที่ 1 — ควบคุมโหลดโดยตรง

การต่อ Sonoff เพื่อควบคุมหลอดไฟหรือปั๊มน้ำขนาดเล็กโดยตรง

Module 4.2: แบบที่ 2 — เพิ่มสวิตช์ Auto / Off / Manual

การเพิ่มสวิตช์เลือกโหมดการทำงานเพื่อควบคุมระบบรดน้ำอัตโนมัติ

Module 4.3: แบบที่ 3 — ใช้ร่วมกับ Magnetic Contactor

การใช้ Sonoff ควบคุมโหลดขนาดใหญ่ผ่าน Magnetic Contactor เพื่อความปลอดภัย
